Where did the main window come from?

The general Windows Form program will determine the main window through the following line of code.

Application.Run(new Form1());

In the Chromium Desktop App, the initial state of the application is determined by index.html (called ChromiumDesktopApp1.index.html in the project). In index.html, the main window of the application is determined by a mainWindow node.

<mainWindow... </mainWindow>

You can learn more details about the Application.Run method by visiting Microsoft's documentation.
